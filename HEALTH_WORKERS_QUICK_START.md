## ðŸ”„ Syncing Data

### Automatic Sync

- Happens automatically when:
  - You submit a form while online
  - Phone reconnects to internet
- You'll see "Syncing..." indicator
- Takes 5-30 seconds depending on record size

### Manual Sync

If you see "Pending Sync" badge:

1. Make sure you have internet (WiFi or mobile data)
2. Look at bottom of screen for "Pending Records" box
3. Tap "Sync Now" button
4. Wait for "Synced!" message

### Stuck Sync?

1. Check your internet connection
2. Try switching from WiFi to mobile data (or vice versa)
3. Restart the app
4. If still stuck after 3 tries, contact supervisor

## âš ï¸ Common Issues & Solutions

### Problem: "Saved offline" but it's not syncing

**Solution**:

1. Check if you have internet (WiFi or data)
2. Open "Pending Records" section
3. Tap "Sync Now"
4. If still doesn't work after 5 minutes, restart app

### Problem: Can't find resident name

**Solution**:

1. Try typing last name instead of first name
2. Check spelling - must be exact or similar
3. If still can't find them:
   - Check they're registered in system
   - Contact supervisor for help

### Problem: "Access Denied" or "Wrong Barangay"

**Solution**:

- You can only enter data for your assigned barangay
- Cannot see data from other barangays
- Contact supervisor if you need permission change

### Problem: App won't open or keeps crashing

**Solution**:

1. Force close the app completely
2. Restart your phone
3. Open app again
4. Clear app cache: Phone Settings > Apps > Health Workers > Storage > Clear Cache
5. If still crashing, reinstall app

### Problem: Photos not uploading

**Solution**:

1. Check photo isn't too large (< 5MB)
2. Use JPG or PNG format (not WEBP)
3. Try uploading without photo first, add later
4. Contact IT if persistent

## ðŸ’¡ Pro Tips

### Tip 1: Batch Entry

**Faster**: Fill multiple forms while offline, then sync all at once

- Go to 5-6 residents, fill all their forms
- Back at office, connect WiFi and sync all at once
- Saves time switching networks

### Tip 2: Offline Backup

**Safer**: Sync at end of EVERY day

- Even if internet is slow, let it finish
- Don't turn phone off during sync
- This backs up your work

### Tip 3: Photo Organization

**Cleaner**: Take all photos first

- Visit resident
- Take clear photos of documents
- Then fill form while reviewing photos
- Reduces need to revisit for photo

### Tip 4: Vital Signs

**Accurate**: Use same equipment as last time

- If using BP cuff from barangay center last time
- Use same one this time for consistency
- If none available, note as "Not measured"

### Tip 5: Battery Saving

**Smarter**: Use battery saver mode + WiFi only

- Battery saver when doing field visits
- Only sync on WiFi (not cellular data)
- Saves both battery and data
